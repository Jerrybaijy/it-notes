## 代码块的区别

### 无列表项的代码块

基准文本，下面是直接回车，在第二行，直接添加的代码块
```html
<dl>
  <dt>First Term</dt>
  <dd>This is the definition of the first term.</dd>
  <dt>Second Term</dt>
  <dd>This is one definition of the second term. </dd>
  <dd>This is another definition of the second term.</dd>
</dl>
```

基准文本，下面是直接回车，在第二行，先缩进，再添加的代码块
  ```html
  <dl>
    <dt>First Term</dt>
    <dd>This is the definition of the first term.</dd>
    <dt>Second Term</dt>

  </dl>
  ```
